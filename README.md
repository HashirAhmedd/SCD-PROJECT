# DOCUMENT VAULT

### 1. Clone the Repository

Open **Command Prompt** or **PowerShell** and run:

```bash
git clone https://github.com/HashirAhmedd/SCD-PROJECT.git
cd SCD-PROJECT
```
### 2. Create Virtual Enviroment
```bash
python -m venv venv
```
### 3. Install Packages
```bash
pip install -r requirements.txt
```

### 4. Run Code
```bash
python main.py
```
